# powerbi-ecommerce-dashboard
## Project Overview
This project analyses delivery performance , customer satisfaction , and refund trends using Power BI.
## KPIs
Total Orders
Total Revenue
Average Delivery Time
On-Time Delivery %
Refund Rate %
Average Service Rating
## Insights
Identified peak order hours
Analysed refund trends by platform
Compared delivery performance by category
## Tools Used
Power BI 
DAX
Data cleaning using power query
